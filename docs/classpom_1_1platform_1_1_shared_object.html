pom::platform::SharedObject Class Reference
Platform

Reference to a shared object file (.dll or .so depending on platform) that holds all functions loaded by it. This code is extremely not threadsafe.

Warning
Do not attempt to load the same shared object file more than once, it will not work.

pom::platform::SharedObject::SharedObject ( std::string  soFilename)

Creation of this object will copy a shared object file on disk to {soFilename}.temp, on destruction it will attempt to delete the file. Throws an std::filesystem::filesystem_error exception on file error.

Loads and returns a function inside the loaded shared object. Returns a SOFunction which will automatically update its internals when reload is called. checking SOFunction.valid is a good decision.

void pom::platform::SharedObject::reload ( )

Unloads and reloads the shared object and reloads all associated functions. Throws an std::filesystem::filesystem_error exception on file error. After running this function its a good idea to recheck SOFunction.valid
