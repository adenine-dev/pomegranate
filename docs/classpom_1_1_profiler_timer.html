pom::ProfilerTimer Class Reference
Debug Utilities

A scope based profiler, not meant to be used directly, instad use the POM_PROFILE_SCOPE() and POM_PROFILE_FUNCTION() macros. Very simple, records time when it is constructed, and when it is destructed, prints the difference. Eventually this will use speedscope with this schema

#include <pomegranate/debug/profiler.hpp>
