pom::InputEvent Struct Reference
Platform » Input Events

Contains all data for an input event. This includes the type of the event, and a pointer to the source window, and the event data.

Warning
This uses the tagged union based on type. Attempting to access the wrong event data is undefined behavior in c++. It is recommended to use the helper functions which do checking. However, if you do not want that checking, the felids are public.

#include <pomegranate/platform/inputEvent.hpp>
