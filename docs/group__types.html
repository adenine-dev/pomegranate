Pomegranate Engine
Types
Maths

Pomegranate has various types that make using the maths slightly less verbose.

Matrix Specializations

Matrix specialization, used mostly for convenience.

using pom::maths::mat2 = Matrix< f32, 2, 2 >
 2x2 component floating point matrix.
 
using pom::maths::mat3 = Matrix< f32, 3, 3 >
 3x3 component floating point matrix.
 
using pom::maths::mat4 = Matrix< f32, 4, 4 >
 4x4 component floating point matrix.
 

Fundamental types

These types are used throughout the codebase when a basic type is needed. Probably follow this convention too.

using u8 = uint8_t
 8bit unsigned integer.
 
using u16 = uint16_t
 16bit unsigned integer.
 
using u32 = uint32_t
 32bit unsigned integer.
 
using u64 = uint64_t
 64bit unsigned integer.
 
using i8 = int8_t
 8bit signed integer.
 
using i16 = int16_t
 16bit signed integer.
 
using i32 = int32_t
 32bit signed integer.
 
using i64 = int64_t
 64bit signed integer.
 
using usize = size_t
 platform dependent unsigned size type.
 
using byte = unsigned char
 platform dependent byte type.
 
using f32 = float
 32bit floating point number, IEEE-754 compliant.
 
using f64 = double
 64bit floating point number, IEEE-754 compliant.
 

Vector Specializations

Vector specialization, used mostly for convenience. Has xyzw, rgba, and stpq members Equivelent to the same name in glsl. These do not do swizzling.

using pom::maths::vec2 = Vector< f32, 2 >
 2 component floating point vector.
 
using pom::maths::vec3 = Vector< f32, 3 >
 3 component floating point vector.
 
using pom::maths::vec4 = Vector< f32, 4 >
 4 component floating point vector.
 
using pom::maths::dvec2 = Vector< f64, 2 >
 2 component double precision vector.
 
using pom::maths::dvec3 = Vector< f64, 3 >
 3 component double precision vector.
 
using pom::maths::dvec4 = Vector< f64, 4 >
 4 component double precision vector.
 
using pom::maths::uvec2 = Vector< u32, 2 >
 2 component unsigned int vector.
 
using pom::maths::uvec3 = Vector< u32, 3 >
 3 component unsigned int vector.
 
using pom::maths::uvec4 = Vector< u32, 4 >
 4 component unsigned int vector.
 
using pom::maths::ivec2 = Vector< i32, 2 >
 2 component int vector.
 
using pom::maths::ivec3 = Vector< i32, 3 >
 3 component int vector.
 
using pom::maths::ivec4 = Vector< i32, 4 >
 4 component int vector.
 
using pom::maths::bvec2 = Vector< bool, 2 >
 2 component boolean vector.
 
using pom::maths::bvec3 = Vector< bool, 3 >
 3 component boolean vector.
 
using pom::maths::bvec4 = Vector< bool, 4 >
 4 component boolean vector.
