constexpr

#include <pomegranate/util/misc.hpp>

Returns a compile time stringified name of the underlying type.
typeName<u32> → "unsigned int".
typeName<u32&> → "unsigned int &".

Like std::binary_search but returns an iterator to the value, returns last if the value is not found. Expects all the same things std::binary_search does, additionally T and the iterator value type must be equal comparable.
