Debug Utilities

Various debugging tools built into Pomegranate.

Pomegranate provides several debug functionalities, because Pomegranate is meant to be crossplatform, profiling and debug capabilities are built in, and not reliant on any platform depdendent external tools.

◆ LogLevel

enum class pom::LogLevel
strong

#include <pomegranate/debug/logging.hpp>

Log levels are the specified filter for terminal output, if the current log level is lower than the specified message, then it should not be printed. For debug builds it defaults to DEBUG and for release builds it defaults to ERROR.

Enumerator
ALL 

Outputs all log messages.

TRACE 

Outputs trace messages and above. (Currently equivelent to ALL)

DEBUG 

Output debug messages and above.

INFO 

Output info messages and above.

WARN 

Output warn messages and above.

ERROR 

Output error messages and above.

FATAL 

Output fatal messages and above.
