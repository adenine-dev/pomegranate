Creates a Buffer.

  • usage: Some combination of BufferUsages that this buffer can be used with.
  • access: The memory access of this buffer. If it is BufferMemoryUsage::GPU_ONLY then initialData should be passed and be of size size. initialDataOffset and initialDataSize should both be 0 indicating that the full buffer will be written to.
  • size: The size of the buffer.
  • initialData: If passed the buffer will be initialized with this data.
  • initialDataOffset: If passed the buffer will be created with the initialData at the specified offset.
  • initialDataSize: If passed the size of initialData. If not passed initialData is assumed to be of size size, and initialDataOffset should be 0.
