pom::gfx::CommandBufferVk Class Referencefinal
Graphics » GFX » Vulkan Implementation

Vulkan CommandBuffer, should be created implicitly through Context::createCommandBuffer. Contains a list of vulkan command buffers and fences corresponding to the number of swapchain images.
