template<typename T, u8 N, u8 M>
struct pom::maths::Matrix< T, N, M >

Generic column major matrix type with N columns and M rows.

You probably want to use the matrix specializations, but this can be used independently for larger matrices. The maximum value for M and N is determined by POM_VECTOR_SIZE_TYPE
