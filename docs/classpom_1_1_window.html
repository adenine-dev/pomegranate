◆ shouldClose()

bool pom::Window::shouldClose ( ) const

Returns true if the operating system or user says that this window should close, this should be checked each update and obeyed. The exception being if a window is tied to some state that the user may not want to lose.

◆ getVulkanSurface()

VkSurfaceKHR pom::Window::getVulkanSurface ( VkInstance  instance) const

Returns the vulkan surface from SDL. Only applicable when vulkan is enabled and is the chosen GraphicsAPI for this window, may fail otherwise. Should be freed with vkDestroySurfaceKHR.

◆ getVulkanDrawableExtent()

VkExtent2D pom::Window::getVulkanDrawableExtent ( ) const

Returns the vulkan drawable extent for swapchains. Only applicable when vulkan is enabled and is the chosen GraphicsAPI for this window, may fail otherwise.

◆ pollEvents()

static void pom::Window::pollEvents ( )
static

Polls the platform for events that have happened in the past update, should be called once every update. Events are handled through their source Window's event callback.
