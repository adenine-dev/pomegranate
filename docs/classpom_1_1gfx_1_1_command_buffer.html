static Ref< CommandBuffer > pom::gfx::CommandBuffer::create ( CommandBufferSpecialization  specialization,
u32  countHint = 0 
)
static

Creates a command buffer tied to this context.

  • specialization: Dictates what this command buffer can do.
  • countHint: hint for how many internal command buffers should be created. If not passed will default to the maximum number of frames in flight. It is useful to set this to 1 for one time use command buffers.

Binds an index buffer, the Buffer must have been initialize with BufferUsage::INDEX. type is how the data in the buffer will be read, this is independent from the type of whatever data was written into the buffer. offset is the offset to the first byte from which indicies will be calculated. Requires the command buffer to have been created with CommandBufferSpecialization::GRAPHICS
