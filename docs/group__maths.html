Maths

All mathematical functionality.

Pomegranate implements its own maths functions, currently it is on a "first-needed first-served" basis. So a lot of functionality is missing. This will be changed both as time goes on and when the 1.0 release happens. All basic types are of the format t## where t is the type abbreviation (u for unsigned int, f for floating point, etc) and ## is the size in bits.

#include <pomegranate/maths/maths.hpp>

The size type for vectors is the type used to index into vectors. Default is u8 giving a maximum vector size of 255, it should be kept as small as possible. If you really need larger vectors, then you can change it.
